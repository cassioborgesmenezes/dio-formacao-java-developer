# dio-formacao-java-developer
O que acha de aprender Java do zero e com os melhores experts da DIO? Bora! Nesta formação você aprenderá desde os fundamentos dessa linguagem de programação até o uso de abstrações poderosas como o Spring Framework. Para isso, inicialmente teremos uma imersão que trará toda a base do Java, além do paradigma de Programação Orientada a Objetos (POO), nos três primeiros módulos. Nos módulos seguintes, preparamos uma seleção com alguns dos nossos melhores conteúdos para que você domine o Java e estejam preparados para o desenvolvimento de projetos mais robustos usando o Spring Framework. Pré-requisitos: Lógica de Programação e Pensamento Computacional são sempre bem-vindos ☕
